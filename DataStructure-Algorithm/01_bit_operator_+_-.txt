※ +, -는 비트연산자로 볼 수 없다.
다만 비트연산자를 활용할 때 -가 나오기 때문에 정리한다.

십진법의 +와 -의 방식과 동일한다.
10진법 : 5 + 1 => 6
 2진법 : 101 + 001 => 110

1이 더해지면서 일의 자리가 올라가 110이 되는 것이다.

10진법 : 9 + 1 => 10
 2진법 : 1001 + 0001 => 1010

10진법 : 5 - 1 => 4
 2진법 : 101 - 001 => 100

10진법 : 5 - 2 => 3
 2진법 : 101 - 010 => 011

10진법 : 21 + 3 => 24
 2진법 : 10101 + 00011 => 11000
